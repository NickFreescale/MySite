## 🎯 快速修改指南

### 要修改时间线信息：
1. 打开 `src/config/site.ts`
2. 找到 `timeline` 数组（第 227 行开始）
3. 修改对应条目的 `period` 字段
4. 保存文件

### 要修改职业描述：
1. 打开 `src/components/about/AboutStory.tsx`
2. 找到第 51-56 行的职业描述
3. 直接修改文本内容
4. 保存文件

### 要修改头像：
1. 将新照片放到 `front/public/images/profile.jpg`
2. 如果使用不同文件名，需要修改 `AboutHero.tsx` 中的路径

---

## ⚠️ 注意事项

1. **时间格式统一**
   - 使用格式：`YYYY.MM - YYYY.MM`
   - 例如：`2019.08 - 2023.06`
   - 当前工作使用：`2023.09 - 至今`

2. **修改后记得测试**
   - 运行 `npm run dev` 查看本地效果
   - 确认没有格式错误

3. **type 字段说明**
   - `education`：教育经历（显示蓝色图标）
   - `work`：工作经历（显示紫色图标）
   - `achievement`：荣誉成就（显示黄色图标）

4. **时间线排序**
   - 时间线会按照数组顺序显示
   - 建议按时间从早到晚排列
