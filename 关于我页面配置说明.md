### 3️⃣ **教育与工作经历（AboutTimeline）**
📁 文件：`front/src/config/site.ts`

在 **`timeline`** 数组中配置（约第241行开始）：

```typescript
export const timeline = [
  {
    id: 1,
    type: "education",        // 类型：education（教育）/ work（工作）/ achievement（成就）
    title: "自动化专业 本科",
    organization: "广东技术师范大学",
    location: "广州",
    period: "2018.09 - 2022.06",
    description: "主修自动化专业..."
  },
  // ... 更多经历
]
```

支持的类型：
- `"education"` - 教育经历（🎓）
- `"work"` - 工作经历（💼）
- `"achievement"` - 成就获奖（🏆）
